What is your new favorite beer?

I always wonder what beer I should get at a bar. There are way too many many choices for a casual beer drinker! I usually know just a few out of dozens of choices there. Thus I decided to build a beer recommendation engine based on online user reviews, which were scraped from one of the most popular beer review sites. Running some multithreaded Python web crawlers on an Ubuntu server for a few days, pretty much the entire site was scraped and the data was temporary stored into a MongoDB, including 6,899,254 reviews, 319,483 beer data, and 362,602 user info. After significant data munging, these raw data was transferred to PostgreSQL for machine learning and analysis purposes.

Machine Learning

This beer recommendation engine is built on user-based collaborative-filtering approach. Since the matrix was sparse, cosine similarity was suitable to calculate distance between beers. Then using this pre-computed matrix, it recommends new beers.
